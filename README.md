# Dual Worlds

Dual Worlds is a game I am making for the 30 day Annual Github Game Off 2018

# Tools Used:
Engine: Unity\
Art: Pikopixel & Gimp\
Code: Visual Studio 2017 Community Mac\
Sound FX: BFXR\
Text: Textcraft.net

# Instructions:
Go to next level: SpaceBar\
Shoot: Left Click\
Aim: Mouse\
Movement: WASD\
Jump: SpaceBar\
